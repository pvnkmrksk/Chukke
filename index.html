ಚುಕ್ಕೆ Chukké - Colourful Unique Tag Generator

Create unique identification tags with color quadrants for tracking and identification purposes. Chukké ಚುಕ್ಕೆ means "dots" or "marks" in Kannada, which perfectly represents this tool's purpose: creating unique visual identification tags for tracking or marking items.

How It Works

Chukké generates unique 2×2 color quadrant tags. Each tag maps to a unique identification number using a base-n system. The colors you select become the "alphabet" for your tags. Tags include orientation markers to help identify their proper orientation.

Color Settings

With 4 colors, you can create up to 256 unique tags.
